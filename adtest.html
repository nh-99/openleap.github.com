OpenLeap

Open-source Leap software

The Leap is an incredible new motion-tracking device from Leap Motion. It tracks movements up to 10 Âµm (!) in 8 cubic feet of 3D space. OpenLeap is a community of open-source enthusiasts making software for this wonderful device.

See the GitHub organization Help OpenLeap & charity!

Who?

Started by Owen Versteeg as an open-source initiative for the Leap.

What?

A bunch of open-source applications that use the Leap. One example is DrawPad.

Ooh! I have an idea for an app!

Awesome! Create a repo if you can code, or email OpenLeap if you can't.

Why?

Many Leap developers asked for a good community site where they could share projects and see examples - so here one is!

How do I add a repo?

Just use GitHub's "transfer ownership" function; transfer the repo to OpenLeap.

I don't have a Leap!

Well, not everyone's awesome enough to get one of the awesome top-secret prototypes - but you might be! Apply here.
